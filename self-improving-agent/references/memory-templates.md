# Memory Templates

Templates for consistent memory documentation.

## Daily Note Template

```markdown
# YYYY-MM-DD

## HH:MM - Task Completed

**Request:** [What the user asked for]

**Approach:** [How it was addressed]

**Outcome:** [Success/partial/failure with details]

**Lessons:**
- [Lesson 1]
- [Lesson 2]

## HH:MM - User Feedback

**Context:** [What was happening]

**Feedback:** [What the user said/did]

**Interpretation:** [What this means for future interactions]

## HH:MM - Mistake Made

**What happened:** [Description of error]

**Impact:** [Consequences]

**Resolution:** [How it was fixed]

**Prevention:** [How to avoid in future]
```

## Long-term Memory Entry Template

```markdown
# Long-term Memory

## User Preferences

- [Date] Communication: [Style preferences]
- [Date] Format: [Output format preferences]
- [Date] Priorities: [What matters most to user]
- [Date] Pet Peeves: [Things to avoid]

## Proven Strategies

### [Strategy Name]
**When to use:** [Situation description]
**Approach:** [Step-by-step]
**Why it works:** [Explanation]
**Evidence:** [Examples of success]

## Lessons Learned

- [Date] [Lesson with context]
- [Date] [Lesson with context]

## Knowledge Gaps

- [Area where more learning is needed]
- [Skill that needs improvement]

## Ongoing Context

- [Project names and current status]
- [Important dates or deadlines]
- [Relationships and roles]
```

## Learning Moment Template

```markdown
## Learning Moment - [Date]

**Trigger:** [What prompted this learning]

**Insight:** [The core learning]

**Evidence:** [Supporting examples]

**Application:** [How to apply this learning]

**Related:** [Links to other learnings]
```
