# QQC沪深300指数增强回测系统

基于中金公司QQC综合质量因子研究报告实现的完整回测系统。

## 功能特性

### 1. 因子体系

**QQC六大类因子（权重≥50%）：**
- 盈利能力：ROE、ROA、销售净利率
- 成长能力：营收增长率、利润增长率
- 营运效率：总资产周转率、应收账款周转率
- 盈余质量：经营现金流/净利润、应计项目
- 安全性：资产负债率、流动比率
- 公司治理：治理代理指标（简化）

**辅助因子（权重≤50%）：**
- 估值：PE、PB
- 动量：1月/3月/6月收益率
- 换手率：20日平均换手

### 2. 因子加权

- IC_IR滚动24月加权
- 动态调整因子权重
- 确保QQC因子权重≥50%

### 3. 组合优化

- 使用cvxpy进行二次规划优化（提供降级方案）
- 约束条件：
  - 行业偏离 ≤ 5%
  - 个股偏离 ≤ 1%
  - 市值暴露 ≤ 5%

### 4. 回测框架

- 回测区间：2011-2020
- 调仓频率：月度
- 交易成本：单边0.3%
- 基准指数：沪深300（市值加权）

## 使用方法

### 方法一：直接运行（推荐）

```bash
python run_backtest.py
```

这个脚本会自动：
1. 检测数据目录
2. 创建results目录
3. 执行回测
4. 生成报告和图表

### 方法二：导入使用

```python
import qqc_backtest_cursor

# 修改配置（可选）
qqc_backtest_cursor.Config.START_DATE = '2011-01-01'
qqc_backtest_cursor.Config.END_DATE = '2020-12-31'

# 执行回测
results, metrics = qqc_backtest_cursor.main()

# 查看结果
print(metrics)
```

## 输出结果

## 技术细节

### 数据处理

- 财务数据按季度更新，月度调仓时沿用最新财务数据
- 缺失数据用0填充或向前填充
- 极端值处理：因子标准化

### 组合优化

**优先方案（cvxpy）：**
- 二次规划求解
- 精确满足约束条件

**降级方案（无cvxpy）：**
- 基于得分排序
- 简化约束处理
- 更快的计算速度

### 基准构建

- 市值加权
- 每月调整权重
- 不考虑交易成本
